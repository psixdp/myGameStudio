## 项目概述

千王骰局是一款类 Balatro 的 Roguelike 骰子构筑游戏。核心体验是"出千改写骰子结果"。
技术方案为网页浏览器（HTML/CSS/JS），Jam 级别 MVP。

- **游戏概念**: `design/gdd/game-concept.md`
- **系统索引**: `design/gdd/systems-index.md`

对高风险系统进行代码原型验证，确保设计可行。

| 原型 | 验证目标 | 状态 | 测试数 |
|------|---------|------|--------|

| 出千能力联动原型 | 能力叠加不会导致数值失控 | ✅ 完成 | 100 全通过 |
| 战斗12步结算原型 | 结算流程完整可运行 | ✅ 完成 | 124 全通过 |

**阶段二完成标志**: 所有原型可运行，核心循环可玩。✅ **已完成 (2026-04-04)**

### 阶段二发现

- **能力叠加安全**：全局最大分数=163（7骰全6+贪欲+牌型大师），远低于500上限
- **游戏难度偏高**：蒙特卡洛1000局×3种策略均0%胜率，建议调优R2目标分数或增加初始资源
- **先加后乘公式有效**：64种被动组合×17种骰子配置均无数值失控

### Sprint 6：表现力与沉浸感升级（设计中）

**设计理念**：从"能玩"到"好玩"——通过视觉表现增强"出千"的戏剧性

**核心概念：三段式情绪曲线**

```
投掷 → 盖碗阶段(出千时机) → 揭晓(紧张释放)
```

#### 阶段1: 投掷（期待感）
- [ ] 骰子滚动动画（1-2秒）
- [ ] 碗盖扣上的动画
- [ ] 滚动/扣碗音效
- [ ] 情绪目标：期待

#### 阶段2: 盖碗（出千时机）
- [ ] 碗盖覆盖骰子的视觉表现
- [ ] "透视"消耗品：玩家可以偷看碗内
- [ ] 出千操作：骰子微微移动的细微动画
- [ ] 出千音效：轻微的拨动声
- [ ] 情绪目标：掌控感、刺激

#### 阶段3: 揭晓（释放）
- [ ] 掀开碗盖的动画
- [ ] 亮出骰子结果
- [ ] 计分动画：分数爆炸式增长
- [ ] 胜利/失败的视觉反馈
- [ ] 情绪目标：释放、爽感

#### 技术实现要点
- [x] 新增 `BOWL_COVERED` 游戏状态
- [x] 骰子滚动动画（CSS @keyframes + JS 延迟）
- [x] 碗盖视觉元素（SVG 或 CSS）
- [x] 出千时的骰子微调动画
- [x] 计分数字滚动效果
